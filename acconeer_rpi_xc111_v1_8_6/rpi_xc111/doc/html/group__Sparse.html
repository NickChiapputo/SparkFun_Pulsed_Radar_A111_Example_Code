Execute service one time.

Activates service, produces one result and then deactivates the service. Blocks the application until a service result has been produced. Will not work if a callback has been set and may fail if the service is already active.

void acc_service_sparse_sparse_callback_set ( acc_service_configuration_t  service_configuration,
acc_sparse_callback_t *  callback,
void *  user_reference 
)

Set a callback to receive sparse results.

If a callback is used, sparse results are indicated by calling the function that is set. Within the callback it is only allowed to copy the data to a application memory to allow the best possible service execution. Setting the callback as NULL disables callback operation.

Parameters
[in]service_configurationThe configuration to set a callback for
[in]callbackThe callback function to set
[in]user_referenceA user chosen reference that will be provided when calling the callback
